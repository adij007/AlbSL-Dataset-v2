# Dataset Reference Pack

This folder contains lightweight CSV samples for repository reference and quick testing.

## Files

- `coordinates_sample_500.csv`: first 500 rows from unified coordinates dataset.
- `video_keypoints_sample_300.csv`: first 300 rows from video keypoints extraction.
- `alfabeti_keypoints_sample_200.csv`: first 200 rows from alfabeti keypoints CSV.

## Notes

- These are subsets only (not full training data).
- Use full data from `datasets/processed/` locally for training.
